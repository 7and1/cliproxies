## On-Page SEO Content

### Homepage Internal Copy (300-400 words)

```
CLIProxies.com is the central hub for CLIProxyAPI, the open-source unified AI proxy gateway.
Connect OpenAI, Anthropic Claude, Google Gemini, Alibaba Qwen, iFlow, and Vertex AI through
a single OpenAI-compatible endpoint.

Designed for developers using AI coding tools like Claude Code, Cursor, and Cline, CLIProxyAPI
simplifies AI integration by providing consistent request/response formats across all providers.
Use model prefixes like gpt-4, claude-3-opus, or gemini-pro to route requests intelligently.

Key features include OAuth authentication for using existing ChatGPT Plus, Claude, or Gemini
subscriptions without API keys. Load balancing distributes requests across multiple credentials,
while automatic failover ensures high availability during provider outages.

CLIProxyAPI is self-hosted, meaning your data never leaves your infrastructure. Credentials are
stored locally, and all proxy logic runs on your servers. The project is open source under
Apache 2.0 license, with an active community contributing ecosystem apps.

Get started by generating a config.yaml using our online generator, then install CLIProxyAPI via
Go, Docker, or package manager. Your unified AI gateway will be running in under 60 seconds.
```

### Apps Page Internal Copy (200-300 words)

```
The CLIProxyAPI ecosystem includes 15+ community-built applications across desktop, web, and
command-line interfaces. These apps leverage CLIProxyAPI's unified API to provide seamless
AI integration across platforms.

For macOS users, native menu bar apps like VibeProxy, Quotio, and ProxyPal offer convenient
access to AI capabilities without leaving your workflow. Track quotas, switch accounts, and
manage routing through polished GUI interfaces.

Windows users can use ProxyPilot, a native fork with TUI and system tray integration. Linux
users have access to CCS (Claude Code Switch) for fast account switching and model routing.

Web-based applications like Subtitle Translator demonstrate CLIProxyAPI's OAuth capabilities,
enabling browser-based AI workflows without exposing API keys. Developers building custom
integrations can use the Go SDK to embed CLIProxyAPI directly into their applications.

All apps in this directory are open source and community-maintained. Submit your own project
to be listed and join the growing ecosystem of AI proxy tooling.
```

### Documentation Page Internal Copy (200-300 words)

```
This documentation covers installation, configuration, and usage of CLIProxyAPI across macOS,
Linux, and Windows platforms. Whether you're setting up a local development environment or
deploying to production servers, you'll find comprehensive guides for every scenario.

The quick start guide gets your proxy running in under 60 seconds. Generate a config.yaml
using our web-based generator, install CLIProxyAPI via Go or Docker, and start routing
requests immediately.

Configuration options include provider credentials, model routing rules, load balancing
strategies, and authentication methods. Use API keys for production deployments or OAuth
for development with existing subscriptions.

The API reference documents all endpoints, including the unified /v1/chat/completions
endpoint that works across all providers. Model prefixes like gpt-, claude-, and gemini-
route requests to the appropriate provider automatically.

Advanced topics cover Docker deployment, Kubernetes orchestration, custom provider endpoints,
TLS configuration, and monitoring via the Management API. Troubleshooting guides help
resolve common issues with authentication, networking, and provider-specific errors.
```
